## Overview
This repository contains the source code for my personal developer portfolio.  
The website presents my background, professional experiences, and technical expertise as a Computer Science Engineering student at CPE Lyon and Software Engineer at Siemens Industry Software.

It was built with performance, accessibility, and clarity in mind, using a modern React-based stack and a minimalist design approach.

---

## Tech Stack

**Core Technologies**
- Next.js
- TypeScript  
- Tailwind CSS  
- React  

**Supporting Tools**
- Node.js  
- React Icons  
- Git & GitHub  

---

## Features
- Responsive layout for all screen sizes  
- Light and dark mode support  
- Dynamic grid for projects with modal details  
- Organized and categorized tech stack display  
- Smooth animations and hover transitions  
- Resume access and contact links  
- Optimized image loading via Next/Image
